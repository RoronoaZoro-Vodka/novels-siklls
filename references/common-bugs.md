# 常见病句和Bug案例

来自真实写作自查中发现的错误。

## 句式类

1. **双重否定病句**：`也不不紧张` → `说不上紧张，也说不上不紧张`
2. **外貌矛盾**：`扎着马尾的短发女生` → 马尾和短发不能共存
3. **同句式连续**：`好像……好像……` → 改一个
4. **重复用词**：`有点奇怪……哪里奇怪` → `有点怪……哪里不对`
5. **代词混淆**：`他们`被patch误改为`他` → 注意单复数

## 错别字类

1. **「躺」=「趟」**：`去了一躺` → `去了一趟`
2. **「在/再」混淆**：检测所有「在」字

## 逻辑类

1. **已覆灭势力的错误引用**：引用的宗门已覆灭但仍作为目的地使用
2. **初次觉醒强度控制**：不能写成全场时停，应有限制
3. **OOC行为**：内向角色主动长篇对话

## 段落格式

1. **「他」字重复**：同一句连续两个「他」，删一个
2. **缺主语**：动作链混乱时需明确主语
