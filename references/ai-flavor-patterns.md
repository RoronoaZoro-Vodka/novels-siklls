# AI味替换表

## 高频替换

| AI味 | 更自然 |
|------|--------|
| 闪烁着 | 闪着 |
| 说不清道不明 | 说不清 |
| 忽然/突然 | 直接写动作 |
| 好像/仿佛/似乎 | 删减约50% |
| 但是/不过/然而 | 约50%可直接去掉 |
| 他的/她的 | 上下文清晰时省略 |
| ……的时候 | → ……时 |
| 一阵XXX | 直接写内容 |
| 他开始XXX | 去掉「开始」 |
| 他继续XXX | 去掉「继续」 |

## 句式特征

| AI味 | 修正 |
|------|------|
| 不是A，也不是B | 既不A也不B |
| 像是……一样 | 去掉「一样」 |
| 就像是…… | 去掉「就」 |
| 他感觉到/他意识到 | 直接写动作 |
| 就在这时/正在这时 | 直接写发生的事 |

## 对话标签

| AI味 | 更自然 |
|------|--------|
| XXX说道 | XXX说 |
| XXX问道 | XXX问 |
| XXX回答道 | XXX回了一句 |
| XXX解释道 | XXX解释 |

## 易bug句式

| 模式 | 错误 | 修正 |
|------|------|------|
| 双重否定 | 也不不紧张 | 说不上紧张，也说不上不紧张 |
| 外貌矛盾 | 扎着马尾，短发 | 选一种 |
| 重复用词 | 有点奇怪……哪里奇怪 | 有点怪……哪里不对 |
| 缺逗号 | 过了不到一分钟出来了 | 过了不到一分钟，出来了 |
