# 番茄小说发布格式规范

## 核心要求

- **纯文本格式**，不加任何 markdown 符号
- **可复制直接发布**，不需要额外排版

## 排版细则

### 段落
- 每段 3-4 行以内
- 段落之间空一行
- 首行不缩进

### 对话
- 统一使用双引号 `"..."` 包裹
- 不用 `「」` 或其他引号符号
- 说话标签用冒号：`张三说："内容"`

### 标点
- 省略号……」」（占2字符）
- 破折号」——」（占2字符）
- 杜绝 `！！！` `？？？`
- 全篇统一中文标点

### 章节格式
```
第X章 章节名

正文从这里开始……
（第X章 完）
```

### 不需要的
- ❌ 作者的话、写作说明
- ❌ markdown 符号
- ❌ 英文标点
- ❌ 未完待续、求收藏求推荐
