# 节奏把控 (Pacing Control)

长篇小说的生命线。节奏崩了，写得再好也看不下去。

## 1. 整体节奏模型

### 能量曲线
```
高潮 ████████
     ██    ██
     █ ██  █ ██
平缓 █ █ █ █ █ █
     ─────────────────▶ 章节推进
```

**原则**：大高潮之间要有小高潮和铺垫，不能一直紧绷也不能一直平淡。

### 黄金比例
- 高潮/爆发：30%
- 推进/发展：40%
- 铺垫/日常：20%
- 收尾/过渡：10%

## 2. 章节级节奏

### 每章结构
```
┌─ 开头钩子（100-200字）: 抓住注意力
├─ 主体推进（2000-3000字）: 剧情/战斗/对话
├─ 小高潮或转折（500-800字）: 小爽点/新信息
└─ 结尾钩子（100-200字）: 让读者想点下一章
```

### 三章一循环
- 第1章：抛出冲突或悬念
- 第2章：冲突升级，你来我往
- 第3章：冲突解决/小高潮 + 新悬念
- 不断循环推进

## 3. 卷级节奏

### 每卷节奏模板
```markdown
- 开卷（1-3章）: 引入新场景/冲突
- 发展（4-8章）: 冲突逐步升级，穿插日常和修炼
- 高潮预备（2-3章）: 山雨欲来风满楼
- 高潮爆发（2-4章）: 大战/重大事件
- 余波收尾（1-2章）: 情绪释放，引出下一卷
```

### 节奏变化信号
读者疲倦时 → 切换场景/引入新角色/突发事件
读者太紧张时 → 日常/幽默/感情戏缓冲
读者觉得拖时 → 加快进度/让反派打上门

## 4. 爽点节奏管理

### 爽点密度
| 阶段 | 小爽点间隔 | 中大爽点间隔 |
|------|-----------|------------|
| 开局（前10万字） | 每1-2章 | 每卷1个 |
| 发展期 | 每2-3章 | 每1-2卷1个 |
| 后期 | 每3-4章 | 每卷1-2个 |
| 终局 | 每1章 | 连续高潮 |

### 爽点类型轮换
不要连续用同一种爽点：
- 打脸 → 突破 → 获得宝物 → 收小弟 → 打脸（循环）

### 百万字长篇的起手节奏

对于预计超过 100 万字的作品，前 5-10 章是「慢热期」：

- **第一章应最轻**：只做最基本的引入——世界的一个剪影、主角的状态
- **前 3 章不要出现完整的力量等级表**：让读者通过事件自行感知实力差距
- **第 1 章不要埋 10 万字后才回收的伏笔**：前期伏笔应在当前或下一卷回收
- **配角出场节奏**：第一章主角孤身亮相，第二、三章再逐步引入核心配角
- **情绪基调**：开局不要太压抑，也不要太激昂，平缓起步才撑得起百万字的起伏

违反示例：
- ⛔ 第一章开篇 500 字就把宇宙世界观全交代了
- ⛔ 第一章同时出场四五个有名字的角色
- ⛔ 第一章结尾出现「三年后」这种时间跳跃

正确示例：
- ✅ 第一章：主角早上出门上学，路上看到灵能广告
- ✅ 第一章只出现 2-3 个角色（主角 + 1-2 个核心配角）
- ✅ 第一章不解释力量体系，只展示觉醒者有特权

## 5. 节奏雷区

### 要避免的
- ⛔ 连续5章以上无小高潮
- ⛔ 刚打完大BOSS立刻接日常（要有缓冲）
- ⛔ 修为突破过于频繁（麻木）
- ⛔ 战斗场面拖太长（超过3章）
- ⛔ 回忆杀过多打断节奏

### 要追求的
- ✅ 每章结尾有「翻页动力」
- ✅ 平淡章节比高潮章节短
- ✅ 多重冲突交织（不要解决一个再来一个）
- ✅ 用限时任务增加紧迫感
