# 文风指南 (Style Guide)

确保整本书数百万字的文风统一，不会前后面目全非。

## 1. 叙述基调

### 视角选择
- **第三人称有限视角**：紧跟主角，只写主角知道的信息
- **偶尔上帝视角**：关键节点跳出，交代局势全貌
- **不建议**：第一人称（网络长篇少见）、频繁切换POV

### 时态与语言风格
- 现在时或过去时，全篇统一
- 语言风格定位示例：
  ```
  风格定位：都市热血，爽快明快
  句子特征：短句为主，打斗时更短
  修辞偏好：使用武道术语增强沉浸感
  禁忌：过度文艺、西方翻译腔
  ```

## 2. 描写规范

### 场景描写公式
```
环境（1-2句）→ 氛围（1句）→ 人物出场（2-3句）
```
控制在 5-6 句内，不堆砌形容词。

### 人物描写
- **出场**：外貌 + 气质 + 实力暗示
- **日常**：衣着变化、神态细节
- **战斗**：气势优先于外貌描写

### 战斗描写偏好
- 本小说风格：[选择]
  - A) 详细写一招一式，有板有眼
  - B) 侧重气势和结果，快节奏
  - C) 两者结合，关键战斗详细，普通战斗略写

## 3. 特有词汇库

建立本小说的特有词汇表，全篇统一使用：

```markdown
## 境界名称
| 编号 | 境界名 | 战力描述 |
|------|--------|---------|
| 1 | [自行定义] | [描述] |

## 功法/招式词汇
- [名称]: [效果简述]

## 势力名称
- [名称]: [定位]

## 特殊称谓
- [术语]: [解释]
```

**注意**：同一个概念不要换着名字叫（如「灵力」和「元气」不要混用）。

## 4. 格式规范

### 段落
- 对话段不超过 3 行
- 描写段不超过 5 行
- 适当分段，忌大段文字堆砌

### 标点
- 省略号用 ……（占2字符位置）
- 破折号用 ——（占2字符位置）
- 引号使用双引号 `""`，全文统一。**不用 `「」` 或其他变体**。
- 对话中的引号保持统一风格

### 字数单位说明
- 网文行业的「字」通常指`wc -m`统计的总字符数（包括中文、标点、英文字母、数字、空格、换行）
- 番茄小说发布平台以此为准
- 统计命令：`wc -m chapter-xxx.md`

## 5. 番茄小说格式规范

用于直接发布到番茄小说平台的格式要求：

- **纯文本，不加 markdown 标题符号**（如 `# ## ***`）
- **首行不缩进**，段落之间用空行分隔
- 每段 3-4 行为宜，不超过 5 行
- 章节标题用粗体或居中处理（平台自带功能）
- 章节末尾标注（第X章 完）
- 文件名无特殊要求
- **每章字数规则**：最低≥2000字符（`wc -m` 统计，含标点和空格），无硬性上限
- **字数统计数据源**：使用 `wc -m` 统计总字符数（含标点、空格、换行），不低于2000即可

## 6. AI味特征速查表

写完后检查这些 AI 常见措辞并替换（详细处理流程见 novel-writer 第8节）：

| 避免的措辞 | 更自然的写法 |
|-----------|------------|
| ……的时候 | ……时 |
| 他/她忽然/突然 | 直接写动作 |
| 似乎/好像/仿佛 | 删减约 50% |
| 经过/路过……的时候 | 经过/路过…… |
| 但/不过/然而 | 约 50% 可去掉 |
| 与此同时/就在这时 | 直接写下一件事 |
| 他感受到/他察觉到 | 通过动作体现 |

**核心原则**：能用动作和对话体现的，就不需要心理独白和修饰语。

每次写作前快速扫一眼：

### 不要出现
- ❌ 现代网络流行语（破防了、躺平）
- ❌ 英文混用（除非设定如此）
- ❌ 过度使用"！"和"？？？"
- ❌ 大段心理独白（用行动和对话代替）
- ❌ 作者跳出来说教
- ❌ 重复用词（「顿时」「忽然」「只见」）

### 尽量少用
- ⚠️ 「震惊！」「恐怖如斯！」类烂大街表达
- ⚠️ 每一个角色都「嘴角勾起一抹冷笑」
- ⚠️ 每一个战斗都「快如闪电」

## 8. 百万字长篇的节奏意识

对于预计超过 100 万字的网络小说，前几章的信息密度需要刻意控制：

### 第一章能放的
- 主角的日常状态和性格剪影
- 世界的一个横截面（通过日常细节自然带出）
- 1-2 个极轻的伏笔
- 一个简单的结尾钩子

### 第一章不要放的
- 完整的力量体系等级说明
- 复杂的势力格局和派系介绍
- 几十万字后才回收的深远伏笔
- 同时密集出场多个配角
- 大量世界观设定堆砌
- 过于明显的角色身份暗示

### 放信息的速度参考
- 读者每章能吸收的新信息量有限，不要让任何一章的「信息密度」超过读者能消化的程度
- 「通过事件展示」永远优于「通过对话解释」
- 不确定「该不该放这个信息」时——先不放，后面有的是机会

## 7. 叙事平衡法则（用户明确要求的文风约束）

以下三条是用户反复强调的核心文风要求，写作过程中必须时刻注意。违反任何一条都会导致章节被退回重写。

### 7.1 避免大篇幅纯叙事

即使是单人场景（主角独自行动、修炼、赶路等没有对话对象的章节），也不能连续超过5句纯描述。**每3-5句描述之后，必须插入至少一句主角的以下内容之一：**
- 内心想法（「他心想……」/「他意识到……」）
- 判断推理（「这地方有些不对劲……」）
- 回忆联想（连接过往经历或伏笔）
- 情绪反应（紧张、疑惑、警觉等）

**纯叙事是指**：只有环境描写 + 动作描写，没有主角的任何主观想法或情绪反应。

**错误示例（纯叙事过长）：**
```
林一沿着山路走了半个时辰。路两旁的树木越来越密，树冠遮住了天光。脚下的路越来越窄，最后变成了一条勉强能通行的小径。他跳过一道溪沟，拨开挡路的藤蔓。前面出现了一座石桥，桥面长满了青苔。
```

**正确示例（每3-5句穿插内心活动）：**
```
林一沿着山路走了半个时辰。路两旁的树木越来越密，树冠几乎遮住了全部天光。他放慢了脚步。这样的地形太适合埋伏了——如果有人想在这里动手，他根本来不及反应。他跳过一道溪沟，右手始终没有离开剑柄，直到前方出现了一座石桥。
他站在桥头看了一眼。桥面的青苔很均匀，不像是近期有人走过的样子。但他总觉得有什么地方不对。
```

**例外**：战斗场面中的连续动作描写不受此限，但战斗结束后必须立刻接入主角的状态和反应。

### 7.2 段落长度控制

**每段原则上不超过3-5句**。当写到第4-5句时，必须主动寻找断句位置分段。

- 对话段：不超过3句（或不超过3行）
- 描写段：不超过5句
- 连续超过5句的段落必须拆分

**分段提示信号**：
- 主语更换时 → 换段
- 场景/视角切换时 → 换段
- 从描述进入内心活动时 → 换段
- 从内心活动回到描述时 → 换段
- 对话和动作交替时 → 换段

**特别注意**：连续多段纯叙事（即使每段控制在3-5句内），也会构成第7.1条中的「大篇幅纯叙事」——此时需要在段落之间插入对话或内心独白来打散叙事密度。

### 7.3 对话和内心独白调节节奏

不能连续多章只有描述没有对话。**即使章节主题是单人修炼或独自行走，也至少要安排以下场景之一来穿插对话**：
- 遇到熟人 / 被拦截 / 偶遇
- 回忆中的对话片段
- 以内心独白形式呈现的自我对话
- 远距离传讯（飞剑传书、传讯玉符等）

**参考比例**：
| 章节类型 | 对话/独白占比 |
|---------|--------------|
| 战斗章 | 20-30%（穿插在战斗中） |
| 日常章 | 40-50% |
| 情感章 | 50-60% |
| 过渡章（修炼/赶路） | 至少20% |

## 8. 易病句模式（写后自查重点）

### 介词/结构词遗漏
- ❌ 「陈烈旁边转笔」→ ✅「陈烈在旁边转笔」
- ❌ 「过了不到一分钟出来了」→ ✅「过了不到一分钟，出来了」

### 矛盾修饰
- ❌ 「扎着马尾的短发女生」→ 马尾和短发不能共存

### 重复用词
- ❌ 「有点奇怪……但说不上来哪里奇怪」→ ✅「有点怪……但说不上来哪里不对」

### 同句式连用
- ❌ 「好像……好像……」→ 两句连续用同一句式，改其中一句
- ❌ 「直到……才……」连续两句 → 换一种表达

### 主语混淆
- 同段中多个角色交替活动时，需明确标注每个动作的执行者
- 避免在上一句的主语是A时，下一句的主语未标明就变成了B
