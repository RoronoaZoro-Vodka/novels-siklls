# 伏笔与反转 (Plot Twists)

让读者猜不到下一步，回味时又觉得「原来如此」。

## 1. 伏笔类型

### 按重要性分
| 类型 | 埋藏到回收距离 | 作用 |
|------|--------------|------|
| 小伏笔 | 几章内 | 当前剧情的小悬念 |
| 中伏笔 | 半卷到一卷 | 推动卷级剧情 |
| 大伏笔 | 多卷到全书 | 核心反转、最终BOSS |
| 终极伏笔 | 全书首尾呼应 | 故事主题的升华 |

### 按内容分
- **身份伏笔**：某人真正身份/血脉的暗示
- **物品伏笔**：看似普通的物品实际是神器
- **对话伏笔**：某人无心之言后应验
- **场景伏笔**：某个不起眼的场景后成关键
- **能力伏笔**：早出现的某个能力设定是为后期大战铺垫

## 2. 伏笔技巧

### 埋法
- **三次原则**：重要伏笔至少暗示三次才回收
- **藏于日常**：最有效的伏笔藏在不起眼的日常里
- **真假混淆**：用一个假伏笔掩盖真伏笔
- **分散注意**：在紧张剧情中顺便提一句重要信息

### 提示强度递增
```
第一次: 模糊暗示（读者不易察觉）
第二次: 稍明显（部分读者可能注意到）
第三次: 清晰提示（多数读者会意识到「有情况」）
回收: 揭晓答案
```

### 反例
- ❌ 伏笔埋了500章自己忘了 → 用伏笔表管理
- ❌ 回收太突兀（前面毫无征兆）→ 不符合三次原则
- ❌ 伏笔太多来不及回收 → 控制数量

## 3. 经典反转类型

### 身份反转
- 废柴主角其实是绝世天才
- 路边老者其实是隐世高人
- 最信任的人其实是内奸
- 宿敌其实是恩人（或者反过来）

### 局势反转
- 必死之局出现转机
- 看似胜利其实是陷阱
- 螳螂捕蝉，黄雀在后
- 多方博弈中的站队反转

### 真相反转
- 历史真相与教科书不同
- 所谓正派可能不是绝对正义
- 反派有自己的正义理由
- 主角的金手指有隐藏代价

## 5. 反转设计公式

### 让反转有效的步骤
1. **建立预期**：让读者相信A是对的
2. **埋藏线索**：但提前埋好了指向B的细节
3. **反转时刻**：揭晓B才是真相
4. **回放**：读者回忆起之前的细节，「原来如此！」
5. **后果**：反转带来的剧情推动力

### 反转的质量标准
- 出乎意料但不突兀
- 逻辑自洽不强行
- 推动故事而不是为了反转而反转
- 改变读者对整个故事的看法
