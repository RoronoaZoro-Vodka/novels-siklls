# 对话写作 (Dialogue Writing)

对话是展示角色性格、推动剧情、传递信息的最直接手段。

## 1. 角色语气差异化

每个主要角色需要有独特的说话方式：

### 差异化维度
| 维度 | 举例 |
|------|------|
| 用词习惯 | 文雅vs粗犷、专业术语vs大白话 |
| 句式长短 | 简洁短句vs长篇大论 |
| 语气词 | "哼"、"呵"、"切"、"嗯" |
| 口癖/口头禅 | "有意思"、"不过如此"、"呵，天真" |
| 语速暗示 | 说话节奏通过标点控制 |

### 身份决定语气\n- 强者：话少、随意、居高临下\n- 智者：话中有话、喜欢用反问\n- 莽夫：直来直去、嗓门大\n- 傲娇：嘴上不饶人、行动出卖内心\n- 老江湖：圆滑、喜欢打哈哈\n- 小人物：结巴、不敢大声\n- **内向/话少的角色**：不主动找人聊天，不说大段独白。如果需要传递信息，通过行动或其他角色的对话侧面展现\n- **冷静理性的角色**：不在公开场合表露强烈情绪，不突然找不熟的人说心事

## 2. 对话功能分类

### 信息传递型
- 交代背景/设定
- 解释战斗/能力
- 透露剧情情报
- **注意**：不要让角色变成设定解说机，要在自然对话中带出

### 冲突推进型
- 争吵/对峙
- 挑衅/激将
- 宣战/下通牒
- **重点**：对话本身就是战斗的一部分

### 情感表达型
- 表白/道歉/告别
- 安慰/鼓励
- 誓言/承诺
- **技巧**：短比长好，行动比语言有力

### 幽默调剂型
- 插科打诨（紧张后的放松）
- 队友互损（显示关系亲密）
- **注意**：不能破坏当前场景的情绪

## 3. 潜台词与留白

### 潜台词金字塔
```
表层（说的内容）
  ↓
意图（为什么说这个）
  ↓
真实（没说出口的）
```

### 留白技巧
- 话说到一半，用动作打断
- 以沉默回应
- 用转移话题回避
- 答非所问

## 4. 对话格式规范

```
"直接引语要用引号。"他说。
"动作穿插在对话中，"他端起茶杯，抿了一口，才缓缓道，"能打破单调。"
"省去冗长的说话标签，"——他眼神一冷——"用动作来代替。"
不用标签时：换行即可。
```

- 一段对话不超过 6-8 行，超过要穿插动作/描写
- 避免连续 3 段以上纯对话没有描写
- 情绪激烈时用短句，情绪平缓时用长句

## 5. 战斗对话

- 战斗中对话要短促有力
- 边打边说符合设定
- 嘲讽是战斗的一部分
- 突破时的吼叫要点燃读者

## 6. 群聊/多人对话

- 明确标注说话者
- 用动作区分角色
- 避免"XXX说""YYY说"的单调重复
- 群聊节奏比两人对话更快
