# 四维自查规范

写完一章后强制执行的四个维度的检查。

## 1️⃣ 设定冲突

- 世界观设定与本卷/全局设定是否一致
- 力量体系与之前展示是否一致
- 角色实力与之前展示对得上
- 角色行为符合人设基线（不OOC）
- 新引入的设定有原作线索支撑

## 2️⃣ 前后矛盾

- 本章开头与上一章结尾自然衔接
- 时间线正确（季节/日夜/先后顺序）
- 位置逻辑正确
- 角色状态与上章结尾一致（伤势/修为/装备）
- 新伏笔不覆盖已有伏笔

## 3️⃣ 错别字

- 的/地/得、在/再、他/她/它
- 专有名词正确
- 引号全部为双引号
- 无 markdown 残留
- 省略号……」破折号」——」

## 4️⃣ 啰嗦篇幅

- 同一信息不重复表达
- 多余心理独白（能用动作对话体现则删）
- 多余过渡语（「与此同时」「就在这时」→直接写）
- 连续同句式（「好像……好像……」→改一个）
- 字数 wc -m ≥ 2000

## 报告格式

```
**四维自查报告：第X章**

| 维度 | 检查项 | 结果 |
|------|--------|------|
| 1️⃣ 设定冲突 | [...] | ✅/❌ |
| 2️⃣ 前后矛盾 | [...] | ✅/❌ |
| 3️⃣ 错别字 | [...] | ✅/❌ |
| 4️⃣ 啰嗦篇幅 | 字数≥2000（wc -m） | ✅ NNNN字 |

自查结论：通过/未通过
```
