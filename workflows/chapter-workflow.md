# 单章写作流程

每章的标准创作过程。

## 步骤

### 1. 准备
- 加载技能：novel-writer, novel-style-guide, novel-characters, novel-pacing
- 阅读上一章结尾，确认衔接点
- 阅读 plot-tracking.md，了解当前进行中的暗线和伏笔

### 2. 写作
- **开头（100-200字）**：具体场景切入，不从背景介绍开始
- **主体（~1800字）**：对话+描写+推进交替
- **结尾（100-200字）**：轻钩子，自然收束
- **格式**：纯文本，段落空行分隔，对话用双引号

### 3. 硬性约束
- 段落不超过 3-5 句
- 连续不超过 5 句纯叙事，穿插内心活动
- 字数 wc -m ≥ 2000

### 4. 四维自查
1. **设定冲突**：世界观、力量体系、角色人设
2. **前后矛盾**：与上一章衔接、时间线、位置、状态
3. **错别字**：的/地/得、在/再、引号、标点
4. **啰嗦篇幅**：重复信息、多余独白、字数达标

### 5. 去AI味
- 替换高频AI词（似乎/好像/忽然→删减）
- 去掉「的时候」「他的/她的」
- 不可改变剧情和对话内容

### 6. 交付
- 保存 chapter-XXX.md
- 输出自查报告
- 输出主线进度
- 上传到 API
